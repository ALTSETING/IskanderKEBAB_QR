# IskanderKEBAB QR

Оновлена структура проєкту після перенесення логіки з концепції **Restaurant of the Future**:

- `site/` — основний застосунок ресторану (клієнтський інтерфейс + бекенд API).
- `admin/` — точка входу/довідка для адмін-частини; фактична робоча адмінка доступна через `/admin` у Flask.

## Запуск

```bash
cd site
python main.py
```

Потім відкрийте:

- `http://localhost:8000/site?table=5` — сайт для гостей (QR-формат може кодувати це посилання з номером столу).
- `http://localhost:8000/admin` — спрощена адмін-панель для прийому/обробки замовлень.

## API

- `POST /api/orders` — створити замовлення.
- `GET /api/orders` — список замовлень.
- `PATCH /api/orders/<id>` — змінити статус (`new`, `accepted`, `done`).
